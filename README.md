# UEAdminAPI-drogon

## 项目简介
UEAdminAPI 是一个基于 Drogon 框架开发的管理 API 服务，主要用于 第三方登录、多因素认证（MFA）等功能。

### 主要功能
- **多因素认证（MFA）**：支持通过电子邮件和短信发送验证码。
- **第三方登录**：支持多种平台的第三方登录。
- **注册与登录**：支持通过邮箱、手机号、密码等方式注册和登录。

## 安装指南
该项目依赖 Drogon 框架和一些第三方库（如 JWT、Tencent SDK）。请确保已安装以下依赖：
- Drogon 框架
- jwt-cpp 库
- Tencent SDK（用于短信服务）
